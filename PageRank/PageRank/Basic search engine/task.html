Description

Read the information about the network from the Standard Input in the following format: an integer n - specifying how many websites are in the network, then n strings separated by spaces which are the names of the websites, then n by n matrix of doubles representing a link matrix for the internet, and at last a string which is a query for the search, a name of a website to look for. The result of the program should be top 5 matched websites for the query which are determined by this rule: if there is a website with the name which was in the query it goes to the top and the rest are the website sorted according to the page rank in descending order. The damping parameter for calculating the page rank should be 0.5 . If there are fewer than 5 websites on the internet, you should return as many as there are.

Examples

Example 1

Input

5
google facebook vk twitter instagram
0.2 0.0 0.0 0.5 0.5
0.2 1.0 0.0 0.0 0.0
0.2 0.0 0.5 0.0 0.0
0.2 0.0 0.5 0.0 0.0
0.2 0.0 0.0 0.5 0.5
instagram

Rank for the websites will be roughly the following

google 21.6
facebook 24.3
vk 16.2
twitter 16.2
instagram 21.6

Output

instagram
facebook
google
vk
twitter

Example 2

Input

3
google facebook vk

1 0.333 0
0 0.333 1
facebook

Rank for the websites will be roughly the following
